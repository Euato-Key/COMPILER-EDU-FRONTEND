# 第一步：文法输入

## 页面功能特色

- **LR文法编辑器**：专门为LR分析设计的文法编辑器
- **增广文法生成**：自动为文法添加增广产生式
- **LR冲突检测**：检测移进-归约和归约-归约冲突
- **文法验证**：确保文法适合LR分析

## 知识背景

LR分析要求：
- 文法必须是上下文无关文法
- 需要增广文法：S' → S
- 避免某些类型的冲突

## 例题演示

原始文法：
```
E → E + T | T
T → T * F | F
F → (E) | id
```

增广文法：
```
S' → E
E → E + T | T
T → T * F | F
F → (E) | id
```
