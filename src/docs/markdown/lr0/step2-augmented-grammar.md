# 第二步：增广文法

## 页面功能特色

- **自动增广**：系统自动为文法添加增广产生式
- **项目集构造**：基于增广文法构造LR(0)项目
- **可视化项目**：直观显示项目集的结构
- **项目编号**：为每个项目分配唯一标识

## 知识背景

增广文法：
- 添加新的开始符号S'
- 添加产生式S' → S
- 便于LR分析的状态构造

LR(0)项目：
- 形如A → α·β的项目
- 点号表示分析进度
- 用于构造状态集

## 例题演示

增广文法：
```
0: S' → ·E
1: S' → E·
2: E → ·E + T
3: E → E· + T
4: E → E + ·T
5: E → E + T·
6: E → ·T
7: E → T·
8: T → ·T * F
9: T → T· * F
10: T → T * ·F
11: T → T * F·
12: T → ·F
13: T → F·
14: F → ·(E)
15: F → (·E)
16: F → (E·)
17: F → (E)·
18: F → ·id
19: F → id·
```
