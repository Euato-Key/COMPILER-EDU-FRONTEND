# 第二步：增广文法

## 页面功能特色

- **自动增广**：系统自动为文法添加增广产生式
- **项目集构造**：基于增广文法构造LR(0)项目
- **Follow集计算**：计算每个非终结符的Follow集
- **项目编号**：为每个项目分配唯一标识

## 知识背景

增广文法：
- 添加新的开始符号S'
- 添加产生式S' → S
- 便于LR分析的状态构造

Follow集计算：
- 用于SLR1冲突解决
- 基于First集计算

## 例题演示

增广文法：
```
S' → E
E → E + T | T
T → T * F | F
F → (E) | id
```

Follow集：
- Follow(E) = {$, +, )}
- Follow(T) = {$, +, *, )}
- Follow(F) = {$, +, *, )}
