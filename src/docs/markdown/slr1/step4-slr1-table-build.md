# 第四步：SLR1分析表构造

## 页面功能特色

- **自动表构造**：基于项目集和Follow集自动构造SLR1分析表
- **冲突解决**：使用Follow集解决移进-归约冲突
- **动作表生成**：生成移进、归约、接受动作
- **转移表生成**：生成状态转移表

## 知识背景

SLR1分析表构造规则：
- 移进动作：对于项目A → α·aβ，在状态i和输入a处填入s j
- 归约动作：对于项目A → α·，在状态i的Follow(A)处填入r k
- 接受动作：对于项目S' → S·，填入acc
