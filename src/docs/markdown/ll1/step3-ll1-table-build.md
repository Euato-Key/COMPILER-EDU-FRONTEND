# 第三步：LL1分析表构造

## 页面功能特色

- **自动表构造**：基于First/Follow集自动构造LL1分析表
- **冲突检测**：自动识别和标记表项冲突
- **可视化表格**：直观显示分析表结构
- **逐步构造过程**：展示每个表项的填充过程

## 知识背景

LL1分析表M[A,a]的构造规则：
- 如果a ∈ First(α)，则M[A,a] = A → α
- 如果ε ∈ First(α)且a ∈ Follow(A)，则M[A,a] = A → α
